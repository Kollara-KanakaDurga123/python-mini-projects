# Python Mini Projects

This repository contains beginner-friendly Python projects that I have created to practice Python basics like loops, conditional statements, and operators.

## Projects

### 1. Number Guessing Game
- A game where the user guesses a random number between 1 and 10.
- Folder: `games/`
- How to run: `python games/number_guessing_game.py`

### 2. Simple Calculator
- A console-based calculator that performs basic arithmetic operations.
- Folder: `calculator/`
- How to run: `python calculator/simple_calculator.py`
