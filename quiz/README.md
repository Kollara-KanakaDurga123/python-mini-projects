# Quiz Application (Python)

A simple console-based Quiz Application developed using core Python concepts.  
This project allows users to answer multiple-choice questions and displays the final score at the end.

---

## Features
- Multiple-choice questions
- User input validation
- Score calculation
- Console-based interaction

---

## Concepts Used
- Lists
- Dictionaries
- Loops
- Conditional Statements
- User Input Handling
